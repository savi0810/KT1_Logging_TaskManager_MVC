# KT1_Logging_TaskManager_MVC

я написал taskmanager с помощью mvc. Не понравилось, там всё слишком шаблонно и завязано на названиях классов файлов,это конечно довольно просто но мне не зашло. По бд у меня sqlite 4 та БД таблички никак не связаны, записи добавляются в другие таблицы в результате действий с таблицей currenttasks (это можно сказать основная таблица, где хранятся текущие задачи). Ef core супермегаудобная вещь. Ну и для логов у меня serilog, просто мне больше нравится да и эта библиотека одна из самых используемых в c#. В прошлом году  я делал логи для тестового проекта и это было очень геморройно, потому что тесты запускаются многопоточно и пишутся в разные файлы. В этот раз всё очень удобно прекрасно)
